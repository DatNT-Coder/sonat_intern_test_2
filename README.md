## Current Features
- Applied Singleton design pattern
- Using DOTween for smoother animations
- Tap to remove blocks
- Block movement blocked by other blocks
- Simple Win / Lose UI and Settings UI
- Multiple level layouts
- Basic particle effect when block is removed

## Possible Improvements
- New block types with special behaviors
- Sound effects and improved game feel
- Mobile UI optimization
